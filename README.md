## 🎮 Projeto Jogo Piano 🎹
Desafio de projeto proposto pelo expert Felipe Aguiar da plataforma da DIO (Digital Innovation One). <br>
O objetivo desse projeto é criar um jogo de tocar piano utilizando HTML, CSS e JavaScript.
